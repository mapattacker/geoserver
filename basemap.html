Base Maps¶

To add the create map services to render in other software, we will need to grap the url. To do that, go to GeoServer > Layer Preview > click layer > copy the url as shown below.

Tableau¶

Go to Map > Background Maps > Map Services.


Click on Add > WMS Services. Paste the url in, and give it a name. Once added with no errors, we can go back to Map > Background Maps and add that map service to the map chart into Tableau.
