GeoServer Explorer¶

GeoServer Explorer is a 3rd party plugin offered in QGIS, an open-source GIS software. It makes management of GeoServer much easier than the default GeoServer administration GUI.

Before anything, go to QGIS page to install the software. Do it, its an excellent one!

Installation¶

Go to Plugins
  • Manage & Install Plugins
  • Not Installed tab
  • Search GeoServer Explorer
  • Click Install Plugin

The installed plugin is located at Web > GeoServer > GeoServer Explorer

Configuration¶

To access GeoServer administration, login according with the host, port, username & password. Default username & password is admin & geoserver.

Upload & Publish Layers¶

Prepare the cartography in QGIS for each layer. Now drag the layer from the TOC to the GeoServer Explorer on the right and drop it under layers. Both the layer and style are uploaded to GeoServer.

Now, choose the coordinate system, and also set the style to the one you just uploaded.


Going to GeoServer interface > Layers, you can see that the layers are not enabled yet.


Click on the layer and under Data tab, check Enable.


Also, click Compute from native bounds.


If you have yet to select a style, you can go to the Publishing tab and select from here.
